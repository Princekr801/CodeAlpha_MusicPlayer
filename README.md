# CodeAlpha Music Player Project

This repository contains the source code for the **Task 4: Music Player using JavaScript** assignment for the CodeAlpha Frontend Development Internship.

## Features Included

This project fully meets all criteria outlined in the CodeAlpha internship instructions:
- ✅ **Design**: Built a stunning, responsive music player interface using HTML and vanilla CSS (glassmorphism UI, smooth micro-animations).
- ✅ **JavaScript Audio Control**: Utilizes the HTML5 `<audio>` API for fully functional play, pause, next, and previous track controls.
- ✅ **Track Information**: Dynamically displays the currently playing song title, artist, album art, and duration.
- ✅ **Progress & Volume**: Features a clickable, real-time progress bar synced with the audio `timeupdate` event, and a functional volume control slider linked to `audio.volume`.
- ✅ **Bonus - Playlist & Autoplay**: Includes a fully interactive "Your Playlists" sidebar that lets you pick songs, and an autoplay feature that seamlessly plays the next track when a song finishes.

## How to Run

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Click on any track or use the "Play All" button at the top to start listening!

> **Note:** The audio playback is powered by high-quality, royalty-free MP3 streams provided dynamically through JavaScript to demonstrate complete Audio API capability without hosting large MP3 files in the repository.
